Eng
If you want to run the program, double‑click “Celownik_python_V2.exe” in the dis folder. After launching, a dot will appear in the center of your screen.
If you want to hide or show the dot, press Alt + 1 — the red dot will toggle on and off.
If you want to close the program, press Alt + Q, or right‑click the icon in the system tray and select “Close”.

PL
Jeśli chcesz uruchomić program, dwukrotnie kliknij „Celownik_python_V2.exe” w folderze dis. Po uruchomieniu kropka pojawi się na środku ekranu.
Aby ukryć lub pokazać kropkę, naciśnij Alt + 1 — czerwona kropka będzie się wtedy pojawiać i znikać.
Aby zamknąć program, naciśnij Alt + Q albo kliknij prawym przyciskiem myszy ikonę w zasobniku systemowym i wybierz „Close”.
